## 提示：

    县级以上区划代码：以民政部 2017/02 区划代码为准
    县级以下区划代码：最全代码是民政部 2013/12/31 的数据，14 年以来县级以下只提供变更，可以爬数据 update 13年的数据
    博雅地名分享网：  提供数据，对比民政部的变更，数据比较新，不过是分散在各个页面
